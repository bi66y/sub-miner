# sub-miner
Recon Automation  <br>
Usage: Just run ./recon.sh  <br>
This bash script will collect all the sub-domains passively and give you some useful details about the live domains.

Required Tools: <br>

amass <br>
assetfinder <br>
httpx <br>
dnsx <br>
nrich <br>
findomain (optional) <br>
sublist3r <br>
subfinder
